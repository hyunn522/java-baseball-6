## 구현 기능 목록

### 게임 시작

- [ ] 게임 시작
- [ ] 3자리 랜덤 난수 생성

### 게임 진행

- [ ] 사용자 입력 숫자 받기
- [ ] 사용자 입력 숫자과 랜덤 난수를 각 자릿수별로 비교
    - [ ] 같은 수가 같은 자리에 있으면 스트라이크
    - [ ] 다른 자리에 있으면 볼
    - [ ] 같은 수가 전혀 없으면 낫싱
- [ ] 비교 결과 출력
- [ ] 사용자가 맞힐 때까지 반복

### 게임 종료

- [ ] 게임 종료 시 입력에 따른 재시도/종료 처리
    - [ ] 재시도 입력 시 게임 시작 단계부터 재실행
    - [ ] 종료 입력 시 게임 종료

## 예외 처리 목록

### 사용자 입력 숫자

- [ ] 3자리가 아닌 경우
- [ ] 숫자로만 구성되지 않은 경우
- [ ] 중복된 수가 존재하는 경우
- [ ] 0보다 작거나 같은 숫자가 존재하는 경우

### 종료 시 입력

- [ ] 게임 종료 시 주어진 옵션(1, 2)과 다른 경우
